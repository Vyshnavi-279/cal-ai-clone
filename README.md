# 🍎 Cal AI Clone

An AI-powered calorie tracking web app.

## Features

- Upload meal images
- AI nutrition analysis
- Estimated calories
- Protein, carbs, and fat breakdown
- Health score and suggestion

## Tech Stack

- Python
- Flask
- Google Gemini API
- HTML/CSS

## Run Locally

```bash
pip install -r requirements.txt
python3 app.py
