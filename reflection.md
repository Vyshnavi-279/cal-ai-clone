# Reflection

This project helped me understand how AI APIs can be integrated into web applications.

I learned:
- Flask routing
- File uploads
- Git and GitHub
- Environment variables
- Gemini API integration
- Debugging API errors

The most difficult challenge was fixing the Gemini model compatibility issue and understanding how Flask handles uploaded files.

If I had more time, I would add user accounts, nutrition history, and a mobile-friendly UI.
